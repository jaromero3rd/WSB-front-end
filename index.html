First we look at posts

We take posts from stock geared subreddit and find all mentions of companies and stocks.

Icon

Saving the data

We take all the posts from the internet and put them into a structured txt file.

Icon

Extrapolating data

We then use Google's cloud Natural language API and find the sentiment of each post and post title. Once we have that we calculate the sentiment score for each ticker (stock' short name).

Reason for creation

Let's be honest hedge funds already have the data i'm going to be giving away on this website. However, we want to empower everyone with this new publically available data set. This will democritize trading strategies that were once before only possible for larger organizations.
